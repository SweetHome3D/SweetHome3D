Aide Sweet Home 3D > Création de logements, de murs et de meubles
     Enregistrement et ouverture d'une logement
 

Pour enregistrer un logement, choisissez Fichier > Enregistrer ou Fichier > Enregistrer sous....
Chaque fichier Sweet Home 3D porte l'extension .sh3d. Si vous ne tapez pas cette extension, Sweet Home 3D l'ajoutera automatiquement au nom du fichier.
Une fois qu'un logement est enregistré, le nom de son fichier apparaît dans la barre de titre de la fenêtre du logement et dans le menu Fichier > Ouvrir le logement récent.

Vous pouvez aussi:

  • exporter le plan du logement dans un fichier au format SVG (Scalable Vector Graphics), en choisissant le menu Plan > Exporter au format SVG...,
  • créer une image au format PNG de la vue 3D en choisissant le menu Vue 3D > Créer une photo...,
  • exporter la vue 3D d'un logement au format OBJ en choisissant le menu Vue 3D > Exporter au format OBJ.... Cette fonctionnalité vous permet de de choisir le fichier OBJ décrivant tous les objets de la vue 3D, avec son fichier MTL et les images de texture stockés dans le même dossier.

Pour ouvrir un logement existant, double-cliquez sur son fichier, choisissez Fichier > Ouvrir... ou choisissez l'un des fichiers du menu Fichier > Ouvrir le logement récent.

•

Un fichier Sweet Home 3D contient le plan d'un logement, l'image d'arrière plan que vous y avez importée, les caractéristiques de sa vue 3D, et tous les modèles 3D des meubles que vous avez ajoutés à ce logement. Cette fonctionnalité permet d'échanger votre fichier avec des personnes qui n'ont pas le même catalogue.

•

Choisissez Fichier > Enregistrer et compresser pour enregistrer votre logement dans un format compressé entièrement compatible avec le format de Sweet Home 3D par défaut. Cette opération prend plus de temps mais génère un fichier bien plus petit.

•

Choisissez Fichier > Ouvrir le logement récent > Effacer le menu si vous voulez effacer la liste des fichiers Sweet Home 3D qui apparaît dans le menu Fichier > Ouvrir le logement récent.
