     Modification des lignes et des courbes
 

Vous pouvez modifier la position des lignes avec la souris, après les avoir sélectionnées dans le plan.
Pour sélectionner une ligne dans le plan, cliquez sur celle-ci. Pour sélectionner plusieurs lignes ou plusieurs objets dans le plan, dessinez un rectangle de sélection qui les contient, ou/et cliquez sur ceux-ci en maintenant la touche majuscule enfoncée.
Pour déplacer les objets sélectionnés, positionnez la souris à l'intérieur de la boîte de sélection d'un de ces objets puis glissez-déposez les, ou appuyez sur les flèches du clavier.

Quand une ligne est sélectionnée dans le plan, vous pouvez déplacer n'importe lequel de ses points, à l'aide des indicateurs qu apparaissent à chacun de ses points.

Quand le pointeur de la souris survole l'un de ces indicateurs, celui-ci change d'aspect pour indiquer que vous pouvez glisser-déposer ce point pour le déplacer.
La couleur et le style d'une ligne peuvent être modifiés grâce à son panneau, en double-cliquant sur cette ligne, ou en choisissant Plan > Modifier les lignes... après l'avoir sélectionné.

Dans le panneau de la ligne, vous pouvez modifier l'épaisseur de la ligne sélectionnée, les flèches à chaque extrémité, le style des jointures entre les lignes, le style de pointillé et sa couleur. La dernière option de la liste déroulante Jointures permet de dessiner une courbe plutôt qu'une ligne brisée.

•

Si vous voulez modifier le style d'un ensemble de lignes, sélectionnez ces lignes et choisissez Plan > Modifier les lignes... Les valeurs des attributs différentes d'une ligne à l'autre dans l'ensemble des lignes sélectionnées, apparaîtront vides dans le panneau des lignes. Vous pouvez aussi recopier le style d'une ligne sur d'autres lignes, en copiant celle-ci dans le presse-papier avec le menu Edition > Copier, puis en choisissant le menu Edition > Coller le style après avoir sélectionné les lignes à modifier.

•

Si vous voulez désactiver le magnétisme sur les lignes que vous dessinez, désactivez le magnétisme dans le panneau des préférences ou pendant le déplacement des points, maintenez enfoncée la touche Alt sous Windows, la touche cmd sous Mac OS X ou les touches Majuscule + Alt sous Linux.

•

Si les objets sélectionnés dans le plan contiennent des murs, des pièces, des meubles, des cotes, des lignes et des textes, tous ces objets seront déplacés avec un glisser-déposer ou avec les flèches du clavier. Si la touche Ctrl (ou sous Mac OS X, la touche alt) est enfoncée pendant que vous glissez-déposez ces objets, ceux-ci seront dupliqués.
