     Importation d'une image d'arrière plan
 

Pour importer une image affichée en arrière plan du plan du logement et accélérer le dessin des murs, choisissez Plan > Importer l'image d'arrière plan....
Ce menu affiche un assistant qui vous aide en quelques étapes à choisir le fichier d'une image et la mettre à l'échelle.

1.

Glissez-déposez le fichier d'une image dans le cadre affiché dans l'assistant, ou cliquez sur Choisir image et choisissez un fichier au format BMP, JPEG, GIF ou PNG.

2.

Une fois que l'image est chargée, cliquez sur Continuer.

3.

Définissez l'échelle de l'image en déplaçant les extrémités de la ligne colorée dessinée dans l'image, de telle manière que cette ligne corresponde à une longueur connue. Saisissez ensuite la longueur réelle de cette ligne dans le champ Longueur de la ligne dessinée, et cliquez sur Continuer.

4.

Définissez l'origine de l'image dans le plan, c'est-à-dire le point dans l'image qui correspond au point (0, 0) dans le plan du logement. Cliquez ensuite sur Terminer.

Une fois que l'assitant image d'arrière plan est fermé, votre image apparaîtra sous la grille du plan du logement, et vous pouvez commencer à dessiner les murs de votre logement.

•

Pour définir avec précision l'échelle de l'image d'arrière plan, agrandissez le panneau de l'assistant image d'arrière plan.

•

Eviter d'importer un fichier trop gros.

•

Quand vous importez l'image scannée du plan de votre logement, assurez-vous que la rotation de cette image est correcte. Une rotation d'un seul degré vous fera perdre l'alignement entre les bords de l'image et certains murs.

•

L'unité affichée par défaut dans l'assistant image d'arrière plan dépend de votre pays. Vous pouvez choisir entre le Centimètre, le Mètre, le Millimètre et le Pouce dans le panneau des préférences.
