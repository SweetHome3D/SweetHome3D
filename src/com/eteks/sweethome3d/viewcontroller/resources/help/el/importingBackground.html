     Εισαγωγή εικόνας φόντου
 

Για να εισαγάγετε εικόνα που να εμφανίζεται ως φόντο στο σχέδιο οικίας και να επιταχύνετε τον σχεδιασμό τοίχων, επιλέξτε Σχέδιο > Εισαγωγή εικόνας φόντου....
Αυτό το μενού εμφανίζει έναν οδηγό που σας βοηθάει να επιλέξετε και να τοποθετήσετε σε κλίμακα ένα αρχείο εικόνας σε λίγα βήματα.

1.

Σύρετε και αφήστε το αρχείο εικόνας στο πλαίσιο που εμφανίζεται στον οδηγό ή κάντε κλικ στο Επιλογή εικόνας και επιλέξτε αρχείο BMP, JPEG, GIF ή PNG.

2.

Αφού φορτωθεί η εικόνα, κάντε κλικ στο κουμπί Συνέχεια.

3.

Καθορίστε την κλίμακα της εικόνας μετακινώντας τα τελικά σημεία της έγχρωμης γραμμής που σχεδιάζεται στην εικόνα κατά τρόπο ώστε αυτή η γραμμή να αντιστοιχεί σε ορισμένο μήκος. Στη συνέχεια, πληκτρολογήστε το πραγματικό μήκος της γραμμής στο πεδίο Μήκος σχεδιασμένης γραμμής και κάντε κλικ στο κουμπί Συνέχεια.

4.

Καθορίστε την αρχή της εικόνας στο σχέδιο, δηλ. το σημείο στην εικόνα που αντιστοιχεί στο σημείο (0, 0) στο σχέδιο οικίας. Στη συνέχεια, κάντε κλικ στο κουμπί Τέλος.

Αφού κλείσει ο οδηγός εικόνας φόντου, η εικόνα σας θα εμφανίζεται πίσω από το πλέγμα σχεδίου οικίας και μπορείτε να αρχίσετε να σχεδιάζετε τους τοίχους της οικίας σας.

   
Πριν τον σχεδιασμό των τοίχων Μετά τον σχεδιασμό των τοίχων της οικίας
και την τοποθέτηση των πορτών και παραθύρων της οικίας

•

Για να καθορίσετε ακριβώς την κλίμακα της εικόνας φόντου, μεγεθύνετε τον πίνακα οδηγού εικόνας φόντου.

•

Αποφύγετε την εισαγωγή πολύ μεγάλων αρχείων.

•

Όταν εισάγετε σαρωμένες εικόνες του σχεδίου της οικίας σας, βεβαιωθείτε ότι οι εικόνες είναι στραμμένες προς τη σωστή κατεύθυνση. Η περιστροφή μίας και μόνο μοίρας μπορεί να οδηγήσει σε απώλεια ευθυγράμμισης μεταξύ ορίων εικόνας και ορισμένων τοίχων.

•

Η προεπιλεγμένη μονάδα μέτρησης που εμφανίζεται στον οδηγό εικόνας φόντου εξαρτάται από τη χώρα σας. Μπορείτε να επιλέξετε Εκατοστόμετρο ή Ίντσα στον πίνακα προτιμήσεις.
