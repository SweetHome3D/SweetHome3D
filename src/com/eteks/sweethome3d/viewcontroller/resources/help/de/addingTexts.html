     Hinzufügen von Texten
 

Um freie Texte im Wohnungsplan hinzuzufügen, müssen Sie zunächst Plan > Text erstellen auswählen. Alternativ können Sie auch das Text erstellen-Werkzeug aktivieren.

 Text erstellen-Werkzeug

Um einen neuen Text einzufügen, klicken Sie die gewünschte Position im Wohnungsplan an und geben Sie anschließend die Beschriftung in dem Bereich ein, der wie unten angezeigt erscheint.

Sobald ein Text angelegt wurde, können Sie seine Beschriftung wieder ändern, indem Sie den Text im Plan doppelt anklicken. Mit den Menüeinträgen im Plan > Schriftart ändern-Untermenü können Sie auch seine größe und seinen Schriftstil anpassen.

Um mit dem Hinzufügen von Texten aufzuhören, wählen Sie Plan > Auswählen oder aktivieren Sie das Auswählen-Werkzeug (oder ein beliebiges anderes Werkzeug).

 Auswählen-Werkzeug

•

Der Name eines Möbelstücks und der Name eines Zimmers kann auch im Wohnungsplan dargestellt werden. Diese Texte, an die Zimmer fläche gebundene Texte sowie Längenangaben von Bemaßungen verhalten sich anders als freie Texte, die mit dem Text erstellen-Werkzeug eingefügt wurden, da ihre Position relativ zu dem Objekt bestimmt wird, an das sie gebunden sind. Daher verschiebt sich solch ein Text mit dem Objekt, an das er gebunden ist, wenn dieses verschoben wird. Wird das Objekt kopiert, wird auch der daran gebundene Text kopiert, und wird es gelöscht, so wird auch der daran gebundene Text gelöscht.

•

Während das Text erstellen-Werkzeug aktiv ist, können einige Werkzeuge und Menüs nicht benutzt werden. Wenn Sie diese Werkzeuge dennoch benutzen möchten, aktivieren Sie wieder das Auswählen-Werkzeug.
