Sweet Home 3D Hilfe > Erstellen von Wohnungen, Wänden und Möbeln
     Wände einzeichnen
 

Um Wände zu zeichnen, müssen Sie zunächst Plan > Wand erstellen oder das Wand erstellen -Tool benutzen.

 Wand erstellen -Tool

Klicken Sie im Plan an den Start-Punkt an dem die neue Wand entstehen soll. Dann klicken oder doppelklicken Sie an die Stelle im Plan, wo sie enden soll. So lange Sie nicht doppel-klicken oder den Escape-Knopf drücken, beendet jeder weitere Klick die eben gezeichnete Wand und beginnt eine Neue am Ende der alten. Während Sie eine Reihe Wände einzeichnen, wird der Startpunkt der neuen Wand an eine bestehende angehängt, wenn Sie auf den Start- oder Endpunkt der bestehenden Wand klicken. Die letzte Wand wird an eine bestehende angehängt, indem Sie an deren Start- oder Endpunkt doppelt klicken. Wenn Sie die Steuerungstaste (Alt unter Mac OS X) gedrückt halten, während Sie den Endpunkt einer Wand anklicken, wird eine runde Wand erzeugt, deren Bogenwinkel von dem Punkt abhängen wird, an dem Sie zum dritten Mal klicken.
Die Klick-Punkte sind die Mitte der Wand bezogen auf ihre Dicke. Die Dicke und die Höhe einer neuen Wand kann im Einstellungspanel geändert werden.
Haben Sie bereits einen Raum gezeichnet, können Sie die Wände um diesen Raum herum erstellen, indem Sie irgendwo in diesem Raum doppelt Klicken.
Wände werden simultan in der 3D-Ansicht und im Plan gezeichnet.

Um das Einzeichnen von Wänden zu beenden, klicken Sie in Plan > Auswählen oder wählen das Auswählen -Tool.

 Auswählen -Tool

•

Falls Sie ein eingescanntes Bild ihrer Wohnung zur Hand haben, importieren Sie es zuerst als Hintergrundbild. Dieses feature beschleunigt die Geschwindigkeit, in der Sie die Wände einzeichnen können.

•

Um Ihnen zu helfen, eine Wand präzise einzuzeichnen, können Sie den Wand-Tooltip und die Ausrichtungslinien benutzen. Drücken Sie die Enter-Taste während der Plan den Fokus hat, werden die Werte in den Tooltips editierbar um Ihnen die Möglichkeit zu geben, die Länge, Winkel und Dicke der ausgewählten Wand zu verändern (Haben Sie noch keine Wand begonnen, dann zeigt der Tooltip die Koordinaten des Startpunktes für eine neue Wand). Benutzen Sie die Tasten hoch, runter, oder Tab um einen anderen Wert zu editieren, und drücken Sie wieder die Enter-Taste um die neuen Werte zu bestätigen. Um wieder die Maus zu benutzen um Wände zu zeichnen, bewegen Sie diese einfach über den Plan.

•

Damit Sie Wände präzise einzeichnen können, benutzen Sie den "Wandlängentooltip" und die Bemaßungslinien.

•

Damit Sie Wände parallel einzeichnen können, sind die Winkel der Wände standardmäßig auf 15° eingestellt. Sie können diesen Magnetismus deaktivieren, wenn Sie Alt-Taste gedrückthalten, während Sie Wände einzeichnen, oder indem Sie den Magnetismus im Einstellungspanel ändern.

•

Die Standardhöhe einer Wand wird dem Wert entnommen, den Sie unter Einstellungen finden. Die Dicke der Wände wird dort ebenfalls eingetragen.

•

Ist eine Wand einmal erstellt, können Sie ihren Ort, Länge, Dicke, Höhe und die Farbe ihrer Textur sowie ihren Glanz verändern.

•

Die Maßeinheit die standardmäßig eingeställt ist, hängt von Ihrem Land ab. Sie können zwischen Zentimeter, Meter, Millimeter und Inch umstellen in den Einstellungen.

•

Wählen Sie Plan > Vergrößern oder Plan > Verkleinern um die Skalierung des Planes zu ändern, oder benutzen Sie das Mausrad, während Sie dabei die STRG-Taste gedrückt halten (oder unter MacOS X die cmd-Taste).

•

Benutzen Sie Luftansicht während Sie Wände einzeichnen. So bekommen Sie einen besseren Überblick über die Wohnung, die Sie gerade erstellen.

•

Während das Wand erstellen -Tool ausgewählt ist, werden einige Werkzeuge und Menüs deaktiviert. Sollten Sie diese jedoch brauchen, wählen Sie wieder das Auswählen -Werkzeug an.
