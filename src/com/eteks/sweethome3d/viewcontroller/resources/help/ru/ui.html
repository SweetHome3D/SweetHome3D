Sweet Home 3D Help
     Пользовательский интерфейс Sweet Home 3D
 

Каждое Each Sweet Home 3D окно для редактирвоания плана состоит из четырех частей с изменяемыми размерами.

1.

Каталог фурнитуры.
Каталог организован по категориям и содержит фурнитуру которую вы можете использовать в вашем дизайне. Вы можете посмотреть содержимое категории кликнув на треугольнике рядмо с категорией или дважды кликнув на категории.

2.

Список фурнитуры дома.
ССписок использованной ыурнитуры для дизайна текущего дома. Список можно сортировать кликнув на заголовке колонкиили через меню Фурнитура > Сортировать по, и список будет отсортирован. Также можно скрыть/показать нужные колонки через меню Фурнитура > Показать колонки.

3.

План дома.
Здесь покзан план дома вид сверху. В нем удобно рисовать стены и распологать фурнитуру, оперировать имим с помощью мышки или стрелками клаиватуры.

4.

Вид дома 3D.
Показывает дом в трехмерном виде. Вы можете видеть дом как вид сверху, так и от лица визитера.

Каждая панель может получить фокус ввода и все меню Sweet Home 3D работают с панелью которая в фокусе. Между панелями можно переключаться панелями через комбинацию клавишshift + tab, или кликнув на панели мышкой.

Панель инструментов тоже может быть перемещена в удобное для Вас место.
