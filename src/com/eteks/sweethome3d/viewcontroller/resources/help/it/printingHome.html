Help di Sweet Home > Creare case, muri e arredamento

     Stampare una casa
 

Per stampare una casa, scegli File > Stampa....
Questo menù mostra la finestra di dialogo per la stampa relativa al tuo sistema, che ti permetterà di scegliere la stampante, il numero di pagine stampate, la qualità di stampa...

Per impostazione predefinita, Sweet Home 3D stampa la lista arredamento, la piantina e l vista 3D corrente di una casa, usando il formato di carta, i margini e l'orientamento di default.
Se vuoi modificare questi parametri, scegli File > Page Setup stampa....

Nel pannello di impostazione della pagina, puoi cambiare il formato della carte e l'orientamento cliccando sul pulsante Formato pagina. Puoi inoltre scegliere se la lista arredamento, la piantina e la vista 3D di una casa dovrebbero essere stampate o meno. Se non vuoi usarela scala della piantina elaborata automaticamente per contenerla nella pagina, puoi scegliere un'altra scala nel campo Scala.
Puoi inostre scegliere l'intestazione e/o il piè di pagina che apparirà in ogni pagina stampata. I campi Intestazione e Piè di pagine accettano testo e alcune variabili che saranno sostituite con il loro valore in fase di stampa. Sono diponibili 7 variabili:

  • $pageNumber per il numero pagina,
  • $pageCount per il numero totale delle pagine,
  • $planScale per la scala della piantina (Quella elaborata automaticamente oppure quella che hai inserito nel campo Scala),
  • $date per la data di stampa,
  • $time per l'ora di stampa,
  • $name per il nome della tua casa,
  • $file per il percorso del file della tua casa.

Per evitare errori nella scrittura del nome corretto di una variabile, usa il pulsante Variabili visualizzato sotto i campi Intestazione e Piè di pagina. Il carattere $ è riservato per le variabili, per stampare il carattere $ usa $$.

Prima di stampare la tua casa, puoi vedere un'anteprima di stampa, scegliendo dal menù File > Anteprima di stampa....

Nel pannello di anteprima di stampa puoi vedere come una casa sarà stampata pagina per pagina. Per cambiare la pagina in anteprima, clicca sulle frecce sopra il pannello o premi i tasti freccia.

•

La funzionalità di stampa potrebbe non funzionare sotto Linux. Se questo è il caso, scegli File > Stampa in PDF... e stampa il PDF creato con i tuoi strumenti di sistema.

•

La stampa della vista 3D potrebbe non essere disponibile a causa della configurazione limitata del tuo sistema. In questo caso, il check box Stampa la vista 3D sarà deselezionato e disabilitato.

•

La vista 3D di una casa è sempre stampata alla risoluzione di150 dpi , in questo modo la stampa riempie una pagina, eccetto i margini.

•

Puoi inoltre mostrare la scala della piantina sulla carta disegnando una dimensione di una misura standard (1 metro ad esempio).

•

L'impostazione della pagina scelta viene salvata con il file della casa.

•

Il pannello di anteprima di stampa può essere allargato.
