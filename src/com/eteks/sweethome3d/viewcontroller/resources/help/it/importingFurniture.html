Aiuto di Sweet Home 3D > Importare sfondo e arredamento
     Importare arredamento
 

Per importare nel tuo catalogo arredamento e/o nella piantina di casa un modello 3D esistente, scegli Arredamento > Importa arredamento..., clicca sullo strumento Importa arredamento o trascina il file di un modello 3D file in una finestra di Sweet Home 3D.

 Strumento Importa arredamento

Queste operazioni visualizzano una procedura guidata che ti aiuta se necessario a scegliere il file di un modello 3D e definisce le sue caratteristiche in pochi passi.

1.

Clicca su Scegli modelli e scegli un file in formato OBJ, DAE, 3DS, LWS o un file ZIP contenente un file di questo tipo. Dopo che il modello è stato caricato, clicca su Continua.

2.

Orienta il modello tramite i pulsanti freccia in maniera tale che il davanti rispecchi il davanti del modello 3D, e cosi via per tutte le viste della tabella. A volte alcuni modelli 3D non sono correttamente renderizzati perchè fronte e retro sono invertiti. Se questo è il caso, spunta il checkbox Inverti le facce fronte e retro.
Poi clicca su Continua.

3.

Cambia se necessario il nome, la misura, l'altezza da terra, il colore del modello importato e se questo modello è spostabile e se è una porta o finestra.
Quando un modello 3D è importato nella lista arredamento della casa, spunta il checkbox Aggiungi al catalogo se vuoi importarlo definitivamente all'interno del catalogo arredamento.
Deseleziona il checkbox Mantieni proporzioni se vuoi cambiare la misura, la profondità o l'altezza del modello 3D indipendentemente da ogni altro.
Clicca sul pulsante Cancella colore se, dopo aver cambiato il suo colore, preferisci conservare i colori di default del modello 3D.
Poi clicca su Continua.

4.

Gira il modello 3D con il mouse per ottenere il miglior punto di vista del modello per l'icona che verrà visualizzata nel catalogo arredamento, nella lista arredamento, e piantina. Poi clicca su Fine.

Dopo che la procedura guidata di importazione arredamento è conclusa, il pezzo importato apparirà nel catalogo arredamento e/o piantina e lista arrdamento, in base alle opzioni scelte.
Puoi modificarlo nella piantina come un qualunque pezzo di arredamento aggunto dal catalogo di default.
Il nome del pezzo importato appare in corsivo nel catalogo. Per modificare le caratteristiche di un modello importato tramite la procedura guidata, fai doppio click su di esso, o seleziona Arredamento > Modifica....

•

L'importazione di un modello 3D tramite trascinamento del suo file nella finestra di Sweet Home 3D potrebbe non essere disponibile su Linux.

•

Il catalogo arredamento non supporta l'omonimia in una data categoria. Finchè il nome di un modello 3D importato nel catalogo non è unico nella sua categoria, il pulsante Continua nel passaggio tre della procedura guidata sarà disabilitato.

•

Il campo di testo della lista a discesa Categoria è modificabile nel caso tu abbia bisogno di creare una nuova categoria di arredamento.

•

L'importazione di un modello 3D all'interno del catalogo arredamento o all'interno della casa dipende dal fatto che il catalogo sia o nofocalizzato quando scegli Arredamento > Importa arredamento.... Se la lista arredamento, o la piantina della casa o la vista 3D sono focalizzate quando scegli questo menù, il modello importato apparirà nel catalogo solo se spunti la voce Aggiungi al catalogo, al terzo passaggio della procedura guidata di importazione arredamento.

•

L'unità di misura visualizzata di default dipende dal tuo paese. Puoi cambiare in centimetri, metri, millimetri o pollici nel pannello preferenze.

•

Siccome il catalogo arredamento è condiviso fra tutte le finestre di Sweet Home 3D, Nessuna operazione fatta sul catalogo (i.e. importazione, modifica e cancellazione) può essere annullata.

•

Le caratteristiche dell'arredamento importato nel catalogo sono automaticamente memorizzate nelle preferenze, e il loro modello 3D e i file di icona sono memorizzati in cartelle specifiche di Sweet Home 3D.
