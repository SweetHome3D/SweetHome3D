     自訂房間
 

　　如果要自訂家居模型中房間的位置，您可以在平面圖中將其選中後用滑鼠進行操作。
　　在需要選中的房間上按一下滑鼠可以選中單個房間。如果要選中平面圖中的多個房間，請在其周圍拖曳出一個選框，或者在按住 Shift 鍵的同時依次按一下每一個房間。
　　如果要移動選中的傢俱，請將滑鼠指標移到平面圖中任意一個已選定的傢俱上，然後拖曳到您想要的位置；您也可以使用鍵盤上的方向鍵進行該操作。請注意，房間與其周圍的牆體和門窗之間並無關聯，如果您想讓這些傢俱也跟隨房間一起移動的話，請將它們全部選中。

　　如果平面圖中只有一個房間被選中的話，您可以用滑鼠單獨拖曳房間的每一個角點。如果房間的名稱、面積被設置為可見的話，您也可以分別移動它們的位置。

1, 2, ...

房間的角標示符表示您可以拖曳此處更改該角點的位置。

名稱位置標示符表示您可以拖曳此處更改房間名稱顯示的位置。

面積位置標示符表示您可以拖曳此處更改房間面積顯示的位置。

　　當滑鼠指標位於這些標示符上時，其形狀會發生變化，指示您可以拖曳滑鼠更改相應的點或文字的位置。如果您將角標示符拖到牆體附近，當滑鼠處於按下狀態且“自動吸附”啟用時，您可以使用顯示出的校準線進行對準。
　　使用滑鼠直接更改的房間會在平面圖和 3D 視圖中同步呈現。

　　您還可以使用“自訂房間”對話方塊更改房間的名稱和 3D 屬性，方法是在平面圖中按兩下您想要更改的房間，或者在選中要更改的房間後使用平面圖→自訂房間... 功能表項目。

　　在該對話方塊中，您可以更改選定房間的名稱，以及其在 3D 視圖中的可見性。
　　您還可以在此處分別選擇房間的地面和天花板在 3D 視圖中的可見性，及其顏色或紋理。

　　在房間四周的牆壁，您將可以改變它 選擇房間的名稱，然後是面積要計算及顯示它。
　　您還可以選擇是否應顯示在 3D 視圖中，並設置它們的顏色、 其紋理和其反光度發言和所選房間的天花板。
當選定的房間相接或非常接近一個或多個牆時，房間窗格中還顯示牆邊周圍房間和牆邊底踢腳板章節，它們會讓你選擇一側沿房間或在房間裡的踢腳板，牆上的紋理或顏色，以及其踢腳板添加踢腳板於選項選中。如有必要，選擇自動拆開沿其他附近的房間牆壁的牆，重新計算選項。

•

如果您想要更改多個房間的 3D 屬性或名稱，請將它們全部選中，然後在功能表上選擇平面圖→自訂房間... 。如果這些房間的屬性不盡相同的話，對話方塊中相應的位置會顯示為空白。

•

使用功能表上的平面圖→更改文字樣式子功能表可以更改房間名稱和面積文字的大小和樣式。

•

您可以添加一個點到現有的房間，將滑鼠游標置於應添加新的點位置，然後選擇添加到房間點平面圖的內容功能表中。同樣，你可以在平面圖內容功能表中選擇從房間裡點刪除滑鼠游標下一個房間刪除的點。

•

如果你想要在不同地區的同一房間的地板上設置不同的顏色或紋理 建立 一個房間，每個部分並更改其顏色或紋理。

•

房間的地板高程總是等於的高程 樓層 房間裡屬於 (即 0 如果沒有高程添加到首頁)。如果房間不在最後一層，天花板的高程將等於標高 + 它屬於高程的高度。否則，一個房間的天花板的高度被計算每個點的房間，和每個這些點高度等於最接近點的一堵牆的高度。這樣就可以創建閣樓的天花板，當您在開始設置不同的高度和點或結束點的斜牆。

•

在 3D 視圖中，房間的天花板僅從下方可見，也就是說只有在您使用“虛擬參觀”模式時才會看到天花板。這樣的設計是為了保證您啟用“顯示天花板”後仍然能在“鳥瞰模式”中看見家居模型內部的佈置。

•

如果你想要禁用的啟動的磁性牆交點，您繪製的直線的角度，禁用的磁性在參數設置窗格中或移動點時，按住Alt鍵在 Windows 中下 the cmd鍵在Mac OS X or the Shift + Alt鍵在Linux.

•

如果您在平面圖中選中的內容包含有牆體、房間、傢俱、折線與曲線、標注和文字，那麼所有這些傢俱會在您使用滑鼠拖曳或鍵盤方向鍵時一起移動。拖曳這些傢俱時按住 Ctrl 鍵（Mac OS X 中為 Alt 鍵）可以進行複製的操作。

•

顯示面積的預設單位取決於作業系統的區域設置，但您仍然可在“參數設置”中將其更改為釐米、米、毫米或英寸。 如果您將單位設置為“釐米”，那麼房間的面積會以平方米為單位；如果設置的單位是“英寸”，房間面積的單位則為平方英尺。

•

使用平面圖→放大或平面圖→縮小可以更改平面圖的顯示比例。
