     Editing dimensions
 

You may change the location and the size of dimensions with the mouse.
To change dimensions, select them in the home plan first. To select one dimension in the plan, click on it. To select multiple dimensions (or other objects) in the plan, draw a selection rectangle around them, or click on dimensions while pressing shift key.
To move the selected dimensions, place the mouse pointer on one of these dimensions then drag and drop them, or press the keyboard arrows.

When one dimension is selected in the plan, you may also:

  • move its start and end point, with the size indicator that appear at each end of the selected dimension,
  • change the size of its extension lines, with the extension lines size indicator that appears in the middle of the selected dimension line.

1.

The start point size indicator shows the point and the extension line you can drag to move the start point of the dimension.

2.

The extension lines size indicator shows the point on the dimension line you can drag to resize its extension lines.

3.

The end point size indicator shows the point and the extension line you can drag to move the end point of the dimension.

When the mouse pointer is upon on of these indicators, it changes to indicate you can drag and drop that point to change the matching attribute of the selected dimension. While you drag and drop the mouse, the size the modified dimension and the length it displays is updated in the plan.

•

When only one dimension is selected, you may move it only if the initial location of the mouse pointer is on the dimension line and not on the extension lines size indicator.

•

To help you keep dimensions parallelism, the angle of a modified dimension is by default a multiple of 15°. You may cancel this magnetism by holding the shift key pressed during mouse move, or by disabling magnetism in the preferences pane.

•

If the items selected in the plan contain walls, rooms, furniture, dimensions and texts, all these items will be moved with a drag and drop or the keyboard arrows. If you keep the ctrl key (or under Mac OS X, the alt key) pressed while you drag and drop these items, they will be duplicated.

•

The length displayed in dimensions is expressed by default in the unit of your country. You may switch to Centimeter or Inch in the preferences pane.

•

The height of the font used to draw dimensions text may be increased or decreased by changing the plan scale with Plan > Zoom in or Plan > Zoom out menu items.
