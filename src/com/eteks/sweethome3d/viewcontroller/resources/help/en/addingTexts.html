Sweet Home 3D Help > Creating homes, walls and furniture
     Adding texts
 

To add free texts to the home plan, you must first choose Plan > Add texts or select the Add texts tool.

 Add texts tool

To add a new text, click at its location in the home plan, then enter its text in the pane that will appear as shown below.

Once a text is created, you may change its text again by double-clicking on it in plan. You may also change its size and its style with the menu items available in Plan > Modify text style submenu.

To end adding texts, choose Plan > Select or select the Select tool (or an other tool).

 Select tool

•

The name of a piece of furniture and the name of a room may also be displayed in the home plan. The behavior of these texts as well as the behavior of the texts of the area of rooms and the length of dimensions are different from the one of the free texts created with the Add texts tool, because their location is relative to the object to which they are bound. Thus, unlike a free text, a bound text will move with its object when you'll move its object, it will be copied with its object and it will be deleted with its object.

•

While Add texts tool is selected, some tools and menus are disabled. If you need these tools, select the Select tool again.
