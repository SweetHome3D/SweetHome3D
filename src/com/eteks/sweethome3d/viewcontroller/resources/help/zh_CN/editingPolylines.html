     更改折线及曲线
 

　　选择他们在平面图后，您可以用鼠标编辑折线的位置。
　　要选择平面图中的一条折线，请单击它。若要选择多个折线或多个对象在平面图中，选择在周围绘制矩形，或 / 和按shift键的同时单击它们。
　　若要移动所选的对象，将鼠标指针放一个选定框内的这些对象，然后拖和放它们，或按键盘箭头。

　　当平面图在选定的一条折线时，在每个点选定的折线点上会出现有点的指标，你可能会移动任何折线的点。

　　当鼠标指针位于这些折点指标之上时，它将改变这指标，您可以拖放那点移动它。
　　颜色和折线的样式也可在其窗格中编辑，当出现折点窗格后双击它， 或选择平面图→更改折线...然后选择它。

　　在折线窗格中，您可以更改 所选的折线厚度、在绘制箭头两端、转折点样式、线段的样式和颜色。最后的转折点样式下拉列表中选择允许绘制一条曲线，而不是线。

•

如果您想要修改的样式的一整组的折线，选择这些折线，然后选择平面图→更改折线...。从一条折线到一组折线组是不同的属性的值，将出现在窗格中的转折线样式是空白的。 您也可以复制粘贴折线样式到其他折在线，通过复制贴簿中使用编辑→复制菜单项，然后选择编辑→粘贴样式您想要选择更改的折线。

•

如果你想要禁用启动的磁性为你绘制，禁用磁性在参数设置窗格中或移动点时，按住Alt键在 Windows下、 在Mac OS X 下按cmd键，在 Linux 下按 + Alt键。

•

如果在平面图中所选的对象包含的墙体、房间、对象、标注、折线和文本，所有这些对象将使用拖放或键盘箭头移动。如果你按住Ctrl键 (或 Mac OS X下， alt键)拖拉这些对象时，这些对象将被覆制过去。
