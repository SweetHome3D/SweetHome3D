Sweet Home 3D Help > Creating homes, walls and furniture
     Drawing walls
 

To draw walls, you must first choose Plan > Create walls or select the Create walls tool.

 Create walls tool

Click in home plan at the start point of the new wall, then click or double-click in plan at its end point. As long as you don't double-click or press the Escape key, each new click points the opposite point of the current wall out and the start point of the next wall. While drawing a set of walls, the start point of the first wall will be attached to an existing wall start or end point if you click on that point, and the last wall will be attached to another wall start or end point by double-clicking on that point. The clicked points are at the middle of the thickness of the drawn walls.
Walls are simultaneously drawn in plan and in the 3D view.

To end the drawing of walls, choose Plan > Select or select the Select tool.

 Select tool

•

If you have a scanned image of your home plan at your disposal, import it as a background image in plan first. This feature speeds up the drawing of walls.

•

To help you draw walls precisely, use wall length tooltip and alignment feedback lines.

•

To help you drawing parallel walls, the angle of walls is by default a multiple of 15°. You may cancel this magnetism by holding the shift key pressed during the drawing, or by disabling magnetism in the preferences pane.

•

The default height of walls is set at home creation from the value of New home walls height in the preferences pane, and the default thickness of new walls is set from the value of New walls thickness in that pane.

•

Once a wall is created, you may change its location, its length, its thickness, its height, its colors or its textures.

•

The unit displayed by default in tooltips, plan rulers and plan grid depends on your country. You may switch to Centimeter or Inch in the preferences pane.

•

Choose Plan > Zoom in or Plan > Zoom out to change plan scale.

•

Use default Aerial view while drawing new walls, you'll get a more useful point of view on your home in the 3D view.

•

While Create walls tool is selected, many tools and menus are disabled. If you need these tools, select the Select tool back.
