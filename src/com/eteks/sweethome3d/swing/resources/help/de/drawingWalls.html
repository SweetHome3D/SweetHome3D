Sweet Home 3D Hilfe > Erstellen von Wohnungen, Wänden und Möbeln
     Wände einzeichnen
 

Um Wände zu zeichnen, müssen Sie zunächst Plan > Wand erstellen oder das Wand erstellen -Tool benutzen.

 Wand erstellen -Tool

Klicken Sie im Plan an den Start-Punkt an dem die neue Wand entstehen soll. Dann klicken oder doppelklicken Sie an die Stelle im Plan, wo sie enden soll. So lange Sie nicht doppel-klicken oder den Escape-Knopf drücken, beendet jeder weitere Klick die eben gezeichnete Wand und beginnt eine Neue am Ende der alten. Während Sie eine Reihe Wände einzeichnen, wird der Startpunkt der neuen Wand an eine bestehende angehängt, wenn Sie auf den Start- oder Endpunkt der bestehenden Wand klicken. Die letzte Wand wird an eine bestehende angehängt, indem Sie an deren Start- oder Endpunkt doppelt klicken. Die Klick-Punkte sind die Mitte der Wand bezogen auf ihre Dicke. Wände werden simultan in der 3D-Ansicht und im Plan gezeichnet.

Um das Einzeichnen von Wänden zu beenden, klicken Sie in Plan > Auswählen oder wählen das Auswählen -Tool.

 Auswählen -Tool

•

Falls Sie ein eingescanntes Bild ihrer Wohnung zur Hand haben, importieren Sie es zuerst als Hintergrundbild. Dieses feature beschleunigt die Geschwindigkeit, in der Sie die Wände einzeichnen können.

•

Damit Sie Wände präzise einzeichnen können, benutzen Sie den "Wandlängentooltip" und die Bemaßungslinien.

•

Damit Sie Wände parallel einzeichnen können, sind die Winkel der Wände standardmäßig auf 15° eingestellt. Sie können diesen Magnetismus deaktivieren, wenn Sie Shift gedrückthalten, während Sie Wände einzeichnen, oder indem Sie den Magnetismus im Einstellungspanel ändern.

•

Die Standardhöhe einer Wand wird dem Wert entnommen, den Sie unter Einstellungen finden. Die Dicke der Wände wird dort ebenfalls eingetragen.

•

Ist eine Wand einmal erstellt, können Sie ihren Ort, Länge, Dicke, Höhe und die Farbe ihrer Textur verändern.

•

Die Maßeinheit die standardmäßig eingeställt ist, hängt von Ihrem Land ab. Sie können zwischen Inch (Zoll) und Zentimetern umstellen in den Einstellungen.

•

Wählen Sie Plan > Zoom in or Plan > Zoom out um die Skalierung des Planes anzupassen.

•

Benutzen Sie Luftansicht während Sie Wände einzeichnen. So bekommen Sie einen besseren Überblick über die Wohnung, die Sie gerade erstellen.

•

Während das Wand erstellen -Tool ausgewählt ist, werden einige Werkzeuge und Menüs deaktiviert. Sollten Sie diese jedoch brauchen, wählen Sie wieder das Auswählen -Werkzeug an.
